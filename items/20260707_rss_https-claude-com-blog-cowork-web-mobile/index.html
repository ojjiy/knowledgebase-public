Claude Cowork is coming to mobile and web

Date: 2026-07-07

Source: rss / Claude

Original URL

Claude CoworkがモバイルおよびWebに対応し、場所を選ばずに作業を継続できるようになりました。この機能は、まずMaxユーザー向けにベータ版として提供が開始され、順次他のプランにも展開されます。 Claude Coworkは、ユーザーがClaudeにタスクを指示すると、ファイル、カレンダー、メール、メッセージアプリ、Web、その他の連携ツールを横断して作業を完了させる機能です。AIエージェントはコード記述で有名になりましたが、Claude Coworkの利用者の90%以上はソフトウェア開発以外の日常的なナレッジワーク（ビジネスオペレーションやコンテンツ作成など）に活用しており、例えば、四半期支出の調整と差異メモの作成、契約書フォルダをリスク flagged の更新トラッカーに変換、会議のトランスクリプトとパイプラインデータからクライアント向けデッキの作成などが行われています。 これまでCoworkはラップトップに限定されていたため、ユーザーが離席すると作業も中断されていましたが、今回のアップデートにより、以下の3点が変更されます。 1. **作業の継続性:** デスクトップで開始したタスクを、スマートフォンで確認し、どこでも完了した出力にアクセスできるようになります。 2. **バックグラウンドでの作業:** ラップトップを閉じたり、会議に出席したりしても、Claudeはバックグラウンドで作業を続行します。スケジュールされたタスクは、デバイスがオンラインでなくても実行されます。 3. **意思決定の保留:** Claudeがユーザーの判断が必要な局面（通話など）に達すると、ユーザーに通知が届き、会議中でも指示を修正すれば、Claudeは新しい指示に基づいて作業を継続します。最終的な承認はユーザーが行います。 デスクトップ版はローカルファイルやブラウザも利用でき、フル機能を提供します。また、Web版とデスクトップ版では、チャットとCoworkが統合され、プロジェクトと成果物が両方で共有されるようになります。 利用開始は、Web版ではclaude.aiのホーム画面から、モバイル版ではiOSおよびAndroidのClaudeアプリのサイドバーからCoworkを開くことで可能です。フル体験にはClaudeデスクトップアプリのダウンロードが推奨されます。 ローンチを記念して、2026年8月5日までCoworkの利用上限が倍増されます。
