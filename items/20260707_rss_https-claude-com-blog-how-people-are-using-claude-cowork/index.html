How people are using Claude Cowork

Date: 2026-07-07

Source: rss / Claude

Original URL

Claude Coworkは、AIを日常業務に活用する新しい方法を示しています。特に、仕事の中核ではないものの、プロジェクトを推進し、ビジネスを円滑に進めるための「付随業務」に役立つことが明らかになりました。 2026年5月に実施された120万件のClaude Coworkセッションの分析によると、最も多く利用されているカテゴリは「ビジネスプロセスとオペレーション」（33.4%）でした。これには、散在する情報を一つのレポートにまとめる、オンボーディングチェックリストを作成する、スプレッドシートを照合するといったタスクが含まれます。このカテゴリは、経理、人事、事務職など、多くの職種に共通する業務であるため、利用率が高いと考えられます。 次に多かったのは「コンテンツ作成とコピーライティング」（16.4%）で、ドラフト作成、スライド資料作成、投稿、提案書作成など、情報合成を要するビジネスコミュニケーション業務に活用されています。これは、マーケティング、広報、事業開発、プロジェクト管理などの職種で広く利用されています。 その他、ソフトウェア開発（8.7%）、DevOpsとインフラ（7%）、リサーチとインテリジェンス（6.4%）、データ分析とビジネスインテリジェンス（5.8%）、ドキュメント処理と抽出（4.1%）、セールスとレベニューオペレーション（4%）などが上位にランクインしました。 Claude Coworkの利用傾向は、開発者がコード作成に特化してClaude Codeを利用するのとは対照的です。Claude Coworkは、専門職が文書作成や事務作業などの付随業務をAIに任せることで、より専門的な業務に集中できるよう支援しています。例えば、弁護士は文書のフォーマットやファイリングをClaude Coworkに任せることで、より複雑な事件に時間を費やすことができます。 AIによる日常業務の支援は増加しており、Claude Coworkは、チーム間の情報共有やアイデア伝達に役立つステータスレポート、プレゼンテーション資料、トラッキングツールなどの作成を支援しています。Claude Coworkの利用方法は進化を続けており、今後もその活用データが公開される予定です。
