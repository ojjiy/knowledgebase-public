マネージャーで止まる人と、部長になれる人の差｜勅使川原晃司

Date: 2026-07-06

Source: rss / Hatena Bookmark Programming

Original URL

この記事は、マネージャーと部長の役割の違いとその昇進に必要な能力について解説しています。 **マネージャーと部長の役割の違い:** * **マネージャー:** 与えられた文脈（目標、戦略、リソースなど）の中で、チームをまとめ、結果を出すことに注力します。プレイヤーからマネージャーへの移行は、「自分でやる」から「人にやってもらう」への変化であり、扱っている仕事の内容は地続きです。 * **部長:** 会社の文脈を自ら作り、推進する役割を担います。ヒト・モノ・カネの制約の中で結果を出しつつ、部門全体の将来を示し、会社を推進する役員候補生と位置づけられます。マネージャーと部長の間の「断絶」は、プレイヤーとマネージャーの間の断絶よりも大きいとされています。部長は、与えられた枠の中で最大化するのではなく、自ら枠を引く側、つまり「問いを立てる」側に移る必要があります。 **部長になるために必要な4つの力:** 1. **ウェットなコミュニケーション力:** ロジックだけでなく、人間関係の力で人を動かす能力。会議室では言えない本音を引き出したり、根回しをしたり、「誰と・どんな場で・何を話すか」を意図を持って設計する力。 2. **やり切る力:** プロジェクト推進に関することなら何でもやり、ボトルネックを自ら特定・除去する力。役割の境界を超えてでも動かす姿勢。 3. **蓋然性を語る力:** 将来を決め、文脈を作り、なぜそこへ行くのかの確からしさを説明する力。ビジョンだけでなく、反対者、費用感、現場の業務変化まで考慮し、論理とビジョンで場を動かす能力。 4. **人の個性を使い分ける力:** 部下を自分のコピーとしてではなく、一人ひとりの強みを読んで役割を設計する力。 マネージャーになるためには、最初の「ウェットなコミュニケーション力」と「やり切る力」で十分ですが、部長になるためには、それに加えて「蓋然性を語る力」と「人の個性を使い分ける力」が不可欠です。 **「蓋然性を語る力」と「人の個性を使い分ける力」の重要性:** * **蓋然性を語る力:** 多くの人は現状への意見は言えても、「どうすればいいのか」という具体的な提案まで落とし込めていません。「蓋然性を語る力」は、単に論理的に説明するのではなく、先を決め、文脈を作り、その確からしさまで示す力です。この力が不足していると、感想を述べるにとどまってしまいます。 * **人の個性を使い分ける力:** 部下を自分の劣化模倣品としてではなく、一人ひとりの強みを理解し、それを活かした役割設計ができる力が必要です。マネージャーの範囲では均質なチームでも回せますが、部長になると部門全体の多様な領域を動かすため、個性を使い分ける力が不可欠です。プレイヤーとして優秀だった人ほど、自分の成功体験を部下に押し付けがちで、この転換が難しい場合があります。 **自己診断と練習:** 筆者は、これらの4つの力を自己診断することの重要性を説き、特に3と4については、できていない人ほど「できている」と思いがちなため注意が必要だと指摘しています。また、「蓋然性を語る力」と「人の個性を使い分ける力」を鍛えるための具体的な練習方法（会議前の一人行脚、決定の逆算、結論ファースト、強み棚卸し、失望リフレーミング、ものさし増設、配役テストなど）を提案しています。 最終的に、部長になるためには、強いプレイヤーとしての結果を出す過程で身につく力に加え、意識して鍛えなければ身につかない「蓋然性を語る力」と「人の個性を使い分ける力」が差を生むと結論づけています。
