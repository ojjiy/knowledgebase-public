Show HN: Slopera, a browser that hallucinates every page with an LLM

Date: 2026-07-09

Source: rss / Hacker News LLM

Original URL

「Slopera」は、LLM（大規模言語モデル）によって生成される「幻覚（hallucination）」を特徴とするデスクトップブラウザです。このブラウザは実際のウェブサイトにアクセスせず、ユーザーが入力したURL（実在するものでも架空のものでも可）に基づいて、そのページの内容（テキストや画像）をその場で生成します。 Sloperaには「レンズ」と呼ばれる機能があり、これにより生成されるウェブページのスタイルを切り替えることができます。 * **Straight（ストレート）**: 現実のウェブサイトに似た、もっともらしい架空の内容を生成します。 * **Extra slop（エクストラ・スロップ）**: アルゴリズムによって過度に最適化されたインターネットを模倣し、ありえない商品、数えきれなくなるリスト記事、過剰なCTAボタンなどを生成します。 * **1998**: Geocities時代のウェブサイトのような、テーブルレイアウト、ヒットカウンター、「Netscape Navigator 4で表示推奨」といった要素を持つページを生成します。 * **Childlike（子供っぽい）**: 6歳児がクレヨンで描いたようなスタイルでページを生成します。 ユーザーは設定画面で独自のレンズを作成することも可能です。各レンズは生成されたページをキャッシュするため、レンズを切り替えることで同じURLを異なる「現実」で再現して閲覧できます。 Sloperaの利用には、Anthropic（Claude）またはOpenRouterのAPIキーが必要です。画像生成にはfal.aiのAPIキーがオプションで必要となります。これらのAPIキーはOSのキーチェーンに暗号化されて保存され、安全に管理されます。 生成されるページや画像は、APIコールとして課金されます。ブラウザ自体や、一度生成されてキャッシュされたページを閲覧するのに費用はかかりませんが、新しくページを生成する際にはAPI利用料が発生します。生成されるコンテンツはLLMの出力であり、モデレーションされていません。 APIキーの取得方法や、使用するモデルによる「忠実度」「速度」「コスト」のトレードオフについても説明されています。例えば、高速・低コストを重視する場合はClaude HaikuやFLUX schnell、高品質なインタラクティブページを求める場合はClaude OpusやGPT Image 2といったより高価で遅いモデルが推奨されています。 開発者向けの情報として、開発環境のセットアップ方法、ビルド、テスト、パッケージングの方法、リポジトリの構成、ライセンス（MIT）などが記載されています。このブラウザで生成されるコンテンツは現実とは無関係であり、実際のウェブサイトとの類似性は意図されたものです。
