Introducing computer use in Gemini 3.5 Flash

Date: 2026-06-25

Source: rss / DeepMind Blog

Original URL

Gemini 3.5 Flash にコンピューター利用機能が組み込まれ、プラットフォームを横断して対話できるエージェントの構築が可能になりました。この機能は、従来は独立した Gemini 2.5 コンピューター利用モデルのみで提供されていましたが、今回 Gemini 3.5 Flash モデルにネイティブ統合されました。 Gemini は、Search や Maps といった組み込みツールとの連携や関数呼び出しに強みを持っており、今回のコンピューター利用機能の統合により、開発者はブラウザ、モバイル、デスクトップ環境で「見て、推論し、行動する」カスタムエージェントをより確実に構築できるようになります。これにより、継続的なソフトウェアテストや、プロフェッショナルアプリケーションにおける知識作業といった、長期間にわたるエンタープライズ自動化タスクのパフォーマンスが向上します。開発者や企業は、Gemini API および Gemini Enterprise Agent Platform 経由で Gemini 3.5 Flash のコンピューター利用機能を活用できます。 Gemini 3.5 Flash は、コンピューター利用機能を用いて Gemini アプリケーションを分析し、機能のカテゴリ別リストを返したり、自身のドキュメントのアクセシビリティ問題を監査したりします。 ライブ環境で動作するエージェントに対するプロンプトインジェクションのリスクを軽減するため、Gemini 3.5 Flash のコンピューター利用機能では、標的型のアドバーサリアルトレーニングが実施されています。さらに、企業が以下の対応を可能にする2つのオプションのエンタープライズセーフガードシステムが提供されます。 * 機密性の高い、または元に戻せない操作に対して、ユーザーによる明示的な確認を要求する。 * 間接的なプロンプトインジェクションが特定された場合、タスクを自動的に停止する。 これらの機能に加え、セキュアなサンドボックス、人間による確認、厳格なアクセス制御などを組み合わせる「多層防御」アプローチが推奨されています。 顧客は既にコンピューター利用機能で価値を創出しており、Browserbase がホストするデモ環境で機能を試したり、Gemini API および Gemini Enterprise Agent Platform のリファレンス実装とドキュメントから開発を開始したりすることができます。
