Which tokens does a hybrid model predict better?

Date: 2026-06-26

Source: rss / Hugging Face Blog

Original URL

本研究では、Transformerモデルとハイブリッドモデル（Attention層とRecurrent層を組み合わせたモデル）のトークン予測能力を比較し、両者のアーキテクチャの違いに起因する性能差を詳細に分析した。 具体的には、Olmo 3 (Transformer) と Olmo Hybrid (ハイブリッド) という、データ、トークナイザー、学習レシピを可能な限り一致させた7Bパラメータモデルを用いて、各トークンタイプにおける予測精度の差を「ロスギャップ」として定量化した。 分析の結果、ハイブリッドモデルは、名詞、動詞、形容詞といった**意味を持つトークン（コンテンツワード）**や、文脈を追跡する必要がある**代名詞の指示対象**などの予測において、Transformerモデルよりも優れた性能を発揮することが明らかになった。これは、Recurrent層が持つ順次的な情報処理能力や状態追跡能力が、これらのトークンの予測に寄 wirkenしていると考えられる。 一方で、ハイブリッドモデルの優位性がほとんど見られなくなる、あるいは弱まるケースも特定された。一つは、**単純に既出の単語やフレーズをそのまま繰り返すトークン**の予測である。この場合、TransformerモデルのAttention機構が、入力全体から該当するトークンを効率的に参照できるため、強みを発揮する。もう一つは、**閉じ括弧**の予測であり、これはAttention機構のみでも十分な予測が可能であることを示唆している。 さらに、1BパラメータのTransformer、ハイブリッド、純粋なRecurrentモデルを用いた実験では、意味を持つ非繰り返しトークンにおいてはハイブリッドモデルと純粋RecurrentモデルがTransformerを上回り、特にハイブリッドモデルが最も高い性能を示した。一方、繰り返しトークンにおいては、Attention機構を持たない純粋RecurrentモデルはTransformerやハイブリッドモデルに遅れをとった。 これらの結果から、モデルアーキテクチャの比較には、全体的な平均ロスだけでなく、特定の能力をテストする**フィルタリングされたロス**を用いることが有効であることが示唆された。また、ハイブリッドモデルは特にオープンクラスのトークン（意味を持つトークン）において強みを持つ可能性があり、これはRecurrent層の状態追跡能力と関連していると考えられる。 本研究は、トークンレベルでの詳細な分析を通じて、ハイブリッドモデルがTransformerモデルに対して持つ固有の強みと弱みを明らかにし、今後のAIモデル開発におけるアーキテクチャ設計の指針となる知見を提供している。
