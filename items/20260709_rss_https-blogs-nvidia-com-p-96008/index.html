NVIDIA Nemotron Achieves Benchmark-Leading Performance With LangChain Deep Agents Harness

Date: 2026-07-09

Source: rss / NVIDIA AI Blog

Original URL

NVIDIAのNemotron 3 Ultraが、LangChainのDeep Agentsハーネスと連携することで、コストを抑えつつも、既存のクローズドモデルに匹敵する、あるいはそれを凌駕する高性能を実現しました。この連携により、オープンソースモデルとして最高の精度を達成し、より多くのタスクをより高いスループットで完了させることが可能になりました。特に、推論コストは主要なクローズドモデルと比較して10分の1に抑えられています。 この成果は、モデル自体の再学習ではなく、モデルを取り巻く環境、すなわちシステムプロンプト、ツール記述、ミドルウェアなどのエンジニアリングの最適化によって達成されました。これにより、企業はNemotron 3 Ultraを活用して、継続的な評価、迅速な実験、そしてビジネスの様々な領域における専門エージェントの開発を、大幅に低いコストで実行できるようになります。 LangChainは、月間2億回以上のダウンロードを誇るエージェントエンジニアリングプラットフォームであり、NVIDIA Nemotron 3 Ultraに特化したDeep Agentsハーネスのチューニングにより、高パフォーマンスなエージェントを実現しました。これにより、企業はカスタマイズ可能で、自社で所有・運用できる完全にオープンなスタックを手に入れることができます。 LangChainの共同創業者兼CEOであるHarrison Chase氏は、「より優れたエージェントを構築する最善の方法は、モデルを取り巻くシステムを改善し続けることです。メモリ、ツール利用、評価、そしてモデルの振る舞いは、チームがそれらをまとめて調整できる場合に相乗効果を発揮します。NVIDIAとの協力は、企業がオープンスタックから強力なパフォーマンスを得つつ、構築するエージェントシステムを管理できることを示しています。」と述べています。 Abridge、Amdocs、Boxといった企業は、既に専門エージェントを自社プラットフォームに組み込んでおり、グローバルシステムインテグレーターであるEYは、LangChain Deep Agents向けのNVIDIA NemoClawブループリントを活用して、NVIDIAの実装能力を拡大し、クライアントが高価値ワークフロー全体で専門エージェントをカスタマイズ、評価、ガバナンスすることを支援しています。 NVIDIA Nemotron 3 Ultraは、LangChainのDeep Agentsベンチマークスイートでテストされ、その実行トレースが分析されました。モデルの再学習ではなく、ハーネスのチューニングに焦点を当てることで、この性能向上が実現しました。LangChain Deep AgentsとNemotron 3 Ultraを使用するすべての開発者は、このチューニングされたプロファイルをLangChainから直接利用できます。 「NVIDIA NemoClaw for LangChain Deep Agents」は、企業が独自の専門AIシステムを構築するためのオープンなリファレンスブループリントです。これは、Nemotron 3 Ultra向けにチューニングされたLangChain Deep Agentsコードと、エージェントアクションを安全に実行するためのNVIDIA OpenShellセキュアランタイムを組み合わせたものです。オープンなモデル、オープンなハーネス、そしてオープンなセキュアランタイムにより、企業はエンドツーエンドでフルスタックを所有し、自社のビジネスの専門知識に合わせてカスタマイズし、改善し、自社のインフラストラクチャ、クラウド、ガバナンスのどこでも実行できます。 AIアシスタントが質問に答えるだけでなく、コアシステム内でアクションを実行するエージェントへと進化するにつれて、この独自性の重要性は増しています。NVIDIA NemoClaw for LangChain Deep AgentsとチューニングされたNemotron 3 Ultraモデルプロファイルは、現在利用可能です。開発者は、LangChainから直接チューニングされたDeep Agentsハーネスを取得するか、NemoClaw for LangChain Deep Agentsブループリントをスタート地点として、ゼロから専門エージェントを構築できます。 LangChain開発者は、Baseten、Crusoe Cloud、DeepInfra、Fireworks、Nebius、Together AIといったプラットフォームでNemotron 3 Ultraにアクセスでき、本番環境でチューニングされたハーネスを直接利用できます。EYは、このオープンソフトウェアスタックを利用して、企業が専門エージェントの構築を開始できるよう支援しています。
