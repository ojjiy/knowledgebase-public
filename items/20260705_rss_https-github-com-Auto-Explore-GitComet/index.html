## GitComet: 高速・ローカルファーストなオープンソースGit GUI GitCometは、高速なGit操作、ローカルファーストなプライバシー、使い慣れたワークフロー、そしてオープンソースの自由度を求めるチームのために開発された、オープンソースのGit GUIクライアントです。Linux、Windows、macOSに対応しています。 ### 主な特徴 * **高速なGit操作:** 巨大なリポジトリやファイル差分を扱う際にも、応答性・機能性を維持することを目指して開発されました。 * **ローカルファーストなプライバシー:** データの大部分をローカルで処理するため、プライバシーが保護されます。 * **使い慣れたワークフロー:** 既存のGitツールに慣れているユーザーでも、スムーズに移行できるようなインターフェースを提供します。 * **オープンソース:** 誰でも無料で利用、改変、再配布が可能です。 * **クロスプラットフォーム対応:** Linux、Windows、macOSで利用できます。 ### インストール方法 各種OS向けに、GitHub Releasesからプリビルドバイナリ/インストーラーをダウンロードできます。 * **Windows:** インストーラーまたはポータブルバイナリ * **macOS:** Homebrew (`brew tap auto-explore/gitcomet && brew install --cask gitcomet`) * **Linux:** * AppImage (Homebrew経由でもインストール可能) * APTリポジトリ (Debian, Ubuntu, WSLg向け) * AURパッケージ (Arch Linux向け) * リリースtarball、.debパッケージ * (aptパッケージ以外をDebian/Ubuntu/WSLgにインストールした場合、GUIランタイムライブラリの別途インストールが必要な場合があります: `sudo apt install libxcb1 libxkbcommon0 libxkbcommon-x11-0`) ### 要件 GitCometは、Git 2.50以降のローカルGitインストールが必要です。 ### エディション（計画中） * **オープンソース (無料):** * ローカルファーストなデスクトップワークフロー * Gitリモート、プル/プッシュ、ステージング、コミット * ワークツリー、ブランチ、履歴全体 * マルチリポジトリブラウジング * インラインおよびサイドバイサイド差分表示 * 2者間・3者間マージツール * **プロフェッショナル (€20 ライフタイムアクセス、早期導入者限定):** * オープンソースエディションの全機能に加え、以下の機能が含まれます: * Claude Code, Codex, GitHub CLI連携 * コードテストカバレッジワークフロー * GitHubおよびAzure DevOps連携 * 早期アクセス期間中の優先的な機能改善 ### 開発者向け情報 * **ソースからのビルド:** `cargo build -p gitcomet --features ui-gpui,gix` * **コントリビューション:** 開発環境設定、ワークスペースレイアウト、テスト、カバレッジに関するドキュメントは `CONTRIBUTING.md` に記載されています。 ### Git difftool / mergetoolとしての利用 GitCometは、`git difftool` および `git mergetool` から呼び出せるスタンドアロンの差分・マージツールとしても利用できます。ヘッドレスモード（アルゴリズムのみ）とGUIモード（インタラクティブなGPUIウィンドウ）の両方をサポートします。 * **設定/アンインストール:** * `gitcomet setup` コマンドで、グローバルにGitCometを差分・マージツールとして設定します。 * `gitcomet uninstall` コマンドで、安全に統合を解除できます。 * `--local` オプションで現在のリポジトリのみを対象にできます。 * `--dry-run` オプションで、実行されるコマンドを確認できます。 * 設定は、GUIが表示可能な場合はGUIツールを、そうでない場合はヘッドレスツールを自動的に選択するように構成されます。 ### CLIモード * **Difftool:** `gitcomet difftool --local <path> --remote <path> [--path <display_name>] [--label-left <label>] [--label-right <label>]` * **Mergetool:** `gitcomet mergetool --local <path> --remote <path> --merged <path> [--base <path>] [--label-local <label>] [--label-remote <label>] [--label-base <label>]` KDiff3やMeldとの互換性も考慮されています。 ### テーマ GitCometは、組み込みテーマとユーザー提供のカスタムテーマをサポートしています。カスタムテーマは、JSONバンドルファイルとして読み込まれます。 ### クラッシュログ クラッシュ発生時、GitCometはクラッシュログを記録し、次回の起動時にGitHub issueとして報告するオプションを提供します。 ### ライセンス GitCometはGNU Affero General
