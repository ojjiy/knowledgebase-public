AWS、開発者向け生成AI「Amazon Q Developer」のサポート終了を告知　各種IDEでの利用は非推奨に

Date: 2026-07-03

Source: rss / AtmarkIT Coding Edge

Original URL

AWS（Amazon Web Services）は、開発者向けの生成AIサービス「Amazon Q Developer」について、2027年4月30日をもってIDE（統合開発環境）プラグインおよびCLI（コマンドラインインターフェース）のサポートを終了することを発表しました。 Amazon Q Developerは、Visual Studio Code、JetBrains、Eclipse、Visual StudioなどのIDEで利用可能でしたが、今後は「Kiro」という新しいサービスに移行されます。Kiroは、仕様主導開発（spec-driven development）の考え方に基づいた新しいIDEプラグインで、AIによるコード生成やテスト、コードのレビューなどを支援します。 今回のサポート終了に伴い、Amazon Q DeveloperのIDEプラグインとCLIは2027年4月30日に利用できなくなります。また、2026年5月15日以降、Builder IDを用いたAmazon Q Developerの利用や、AWS CLIからのAmazon Q Developer CLIの利用はできなくなります。 Amazon Q Developer Proを利用しているユーザーは、2027年4月30日のサポート終了後もKiroに移行して利用を継続できます。KiroはGitHubから無料でダウンロード可能で、AIによるコード生成、テスト、レビューなどを支援する機能を提供します。 Amazon Q DeveloperのIDEプラグインは、IDEのテーマに合わせたUIを提供し、開発者によるインテリジェントなコーディングを支援していましたが、今後はKiroがその役割を引き継ぎます。
