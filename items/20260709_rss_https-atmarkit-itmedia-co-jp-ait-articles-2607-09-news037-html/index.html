社内のWindows環境で「数百のAIエージェント」を隔離実行　NVIDIAとMicrosoftが共同開発したデスクサイドマシンの全容

Date: 2026-07-09

Source: rss / ITmedia AI+

Original URL

NVIDIAとMicrosoftは、社内Windows環境で数千ものAIエージェントを隔離実行できるデスクサイドAIマシン「NVIDIA DGX Station for Windows」を共同開発しました。このマシンは、NVIDIA GB300 Grace Blackwell Ultra Desktop Superchipを搭載し、1兆個のパラメータを持つ大規模AIモデルを処理できます。 NVIDIA DGX Station for Windowsは、AIモデルの開発・実行に必要なGPU、CPU、メモリ、ストレージを統合した高性能なAIワークステーションです。従来、大規模AIモデルはLinux環境で実行されることが一般的でしたが、本製品はWindows環境でもAIモデルを容易に実行できる環境を提供します。 主な特徴は以下の通りです。 * **高性能ハードウェア:** NVIDIA GB300 Grace Blackwell Ultra Desktop Superchipを搭載し、AIワークロードに最適化されています。 * **Windowsネイティブ環境:** Windows OS上でAIモデルを実行できるため、既存のWindows環境との親和性が高く、開発者は使い慣れた環境でAI開発を進められます。 * **隔離実行:** 数千ものAIエージェントを安全かつ独立して実行できるため、セキュリティやリソース管理の面で優れています。 * **NVIDIA OpenShell:** WindowsのGPUリソースをAIモデルが効率的に利用できるよう、NVIDIA OpenShellが提供されます。これにより、AIモデルの実行環境が最適化され、セキュリティリスクを低減します。 この新製品は、AI開発者、データサイエンティスト、ITプロフェッショナルなどが、より迅速かつ効率的にAIモデルを開発・展開できるよう支援します。2026年4月より発売予定です。
