Fable 5を顧問、Sonnet 5を実行役とすることでコストを抑える方法

Date: 2026-07-08

Source: rss / Hatena Bookmark Programming

Original URL

Claude APIやAWS上のClaude Platformにおいて、Claude Fable 5のような高機能モデルを「アドバイザー」として、Claude Sonnet 5のような軽量モデルを「実行役」として連携させることで、コストを大幅に抑えつつ高いパフォーマンスを維持する「Advisor tool」が紹介されました。この手法では、高度な判断や計画立案はFable 5が行い、実際の処理の大部分は安価なSonnet 5が担当するため、トークン料金の大部分が低レートで処理されます。SWE-bench Proのベンチマークでは、Fable 5単独の場合と比較して、約63%のコストで約92%の性能を達成したとのことです。この方法は、コーディングやリサーチのように、計画と実行を分離できる長時間のタスクに適しており、プロンプトキャッシュ機能と組み合わせることで、効率的で高度なワークフローを実現できます。
