後工程で発覚する脆弱性を4割減　「Claude Code」の新プラグインの仕組み、導入方法

Date: 2026-07-07

Source: rss / AtmarkIT Coding Edge

Original URL

Anthropic社は、AIコーディング支援ツール「Claude Code」に、プルリクエスト（PR）のレビューを支援する新プラグイン「Security Guidance」を導入しました。このプラグインは、開発者が作成したコードが、PRでレビューされる前に、潜在的な脆弱性やセキュリティ上の問題を自動的に検出・修正するのを支援します。 このプラグインの主な機能は以下の3点です。 1. **コードの構文チェック:** 開発者が記述したコードの構文エラーや、APIの誤用などを検出します。また、HTML5、DOM API、JavaScriptなどのコード記述における問題点も指摘します。 2. **コードの意図の解釈:** コードが意図した通りに機能するかどうかを、自然言語処理を用いて解釈・分析します。これにより、コードの曖昧さや、意図しない副作用を特定します。 3. **セキュリティレビュー:** コードを分析し、潜在的なセキュリティリスクや脆弱性を特定します。例えば、SQLインジェクションやクロスサイトスクリプティング（XSS）などの脆弱性を検出するのに役立ちます。 このプラグインは、PRでレビューされる前にコードの問題点を修正することで、後工程で発覚する脆弱性を4割削減することを目指しています。 導入方法は、`plugin install security-guidance`というコマンドを実行するだけで、Claude CodeのIDE（統合開発環境）から利用できます。Claude Codeは、GitHub Copilotのような既存のAIコーディング支援ツールと比較しても、開発者の生産性向上に貢献できると期待されています。
