React Profilerを試す

Date: 2026-06-26

Source: rss / Qiita Trend

Original URL

React Developer ToolsのProfiler機能を使用して、Reactコンポーネントの描画パフォーマンスを測定する方法について解説しています。 まず、React Routerを利用したアプリケーションで、意図的に1秒かかる非同期処理を`RootLayout`コンポーネントに追加し、そのパフォーマンスを測定します。`Suspense`コンポーネントを使用して非同期処理の完了を待つUIを実装し、Profilerで確認すると、`Suspense`と`RootLayout`で描画に時間がかかっていることが示されます。 次に、追加した重い処理を削除して再度測定すると、`RootLayout`の描画時間が短縮されていることが確認できます。 結論として、React Developer ToolsのProfilerは、コンポーネントの描画に時間がかかっている箇所を迅速に特定するのに役立つツールであり、「描画がもっさりしている」といった問題が発生した際に活用することを推奨しています。
