MUFG aims to become AI-native with OpenAI

Date: 2026-07-07

Source: rss / OpenAI News

Original URL

MUFGはOpenAIと提携し、約35,000人の従業員を対象にChatGPT Enterpriseの導入を進め、AIネイティブ企業への変革を目指しています。この取り組みは、単なる効率化ツールとしてではなく、人間の思考力と創造性を拡張し、従業員とAIが共に学びながらより良い金融サービスを構築することを目指しています。 2026年初頭から、三菱UFJ銀行では段階的にChatGPT Enterpriseを導入し、従業員が日常業務で活用できるようになりました。MUFGは、リーダーシップによるトップダウンのコミットメントと、従業員が日常業務でAIを活用できるようなボトムアップの支援を組み合わせることで、AIを日常業務の一部とするシステム、環境、文化を構築しています。 OpenAIは、サービス設計から導入までを支援するパートナーとして、MUFGの顧客サービス革新と銀行業務変革を推進しました。特に、ChatGPT Enterpriseの導入にあたっては、その広範な適用性とセキュアな利用が決め手となりました。金融機関としての管理・ガバナンス要件を満たしつつ、多くの従業員が既にChatGPTに親しんでいたことが、円滑な導入を後押ししました。 導入にあたっては、従業員がAIを自信を持って活用できるよう、ツールの理解、業務への組み込み、そして情報管理、承認ルート、利用ルールなどの明確なプロセスが整備されました。OpenAIは、エンタープライズ利用ガイダンス、運用計画、製品教育、従業員向けトレーニングプログラム、カスタムGPTワークショップ、経営層向け学習セッションなどを提供しました。さらに、AI利用前のeラーニングを必須とするなど、安全かつ効果的な利用のための共通基準を確立しました。 この結果、従業員はAIを「一部の専門家だけが使うツール」ではなく、「日常業務で自然に使えるパートナー」として捉えるようになりました。導入から4ヶ月で1,800以上のカスタムGPTが作成され、従業員は「AIバンカー」と呼ばれる、各部署のワークフローに合わせたAIを開発し、定型業務の効率化や専門知識へのアクセス向上に活用しています。これにより、従業員はより価値の高い業務や顧客との直接的な対話に時間を割けるようになっています。 MUFGは、社内変革にとどまらず、AIが顧客の金融サービス体験をどのように変えるかについても探求しています。例えば、資産管理アプリMoneytreeでは、ChatGPT上で自然言語で質問することで、口座残高や取引明細などの情報を確認できるようになります。また、ロボアドバイザーサービスを提供するWealthNaviでは、AIコンシェルジュや、顧客のライフステージに合わせたパーソナライズされた推薦を行うMAP（Money Advisory Platform）などの導入を計画しています。 これにより、MUFGは、従来の銀行アプリや支店中心の体験から、AIとの日常的なやり取りを通じて、金融ガイダンスから取引までを自然に移行できる未来を目指し、より多くの顧客にAIネイティブな金融サービスを提供することを目指しています。
