## 記事要約：M365 Copilot と PowerShell を活用した「ちょいRPA」による業務自動化 本記事では、セキュリティや運用ルールにより自由にツールを使えない現場でも、Microsoft 365 Copilot と PowerShell を組み合わせることで、手軽に業務自動化（RPA）を始める方法を紹介しています。特に、画面操作を伴うUIオートメーションの利点と、その実現方法に焦点を当てています。 ### 1. はじめに * **課題**: 業務自動化のニーズがあるものの、利用できるツールが限定される。 * **解決策**: Microsoft 365 Copilot と PowerShell の組み合わせによる、PowerShell 単体で完結する自動化。 * **UIオートメーションの利点**: 画面の動きで自動化が視覚的に分かりやすい。 * **記事のゴール**: * PowerShell 単体で実現できる現実的な自動化例の提示。 * Copilot への効果的な質問（プロンプト）の理解。 ### 2. 前提・環境情報 * **基本構成**: PowerShell 単体で完結。Excel は発展編で任意追加。 * **使用ツール**: * **必須**: PowerShell、ブラウザ（Edge等）、社内ポータル（経費精算システムなど） * **任意**: Microsoft 365 Copilot / Excel (スクリプトの下書き、ファイル出力、ボタン実行、結果確認に利用) * **Copilot**: M365 Copilot (Basic) または Premium、GPT 5.5 Think Deeper で検証。他のAIでも可。 ### 3. 自動化対象システム * **業務**: 経費精算の状況確認。 * **方法**: AI (Copilot) を活用して、経費精算システムから申請状況を自動で取得するスクリプトを作成。 ### 4. 経理清算状況の確認を自動化 * **Copilot への指示例**: * 「UIautomationでコードを書いて。URLは〇〇。申請状況を確認ボタンを押し、2026年6月 経費精算と入力して検索ボタンを押し、一番上の詳細を見るボタンを押し、ステータスを表示を押して、現在のステータスの下にあるステータスを確認してユーザーに教える。」 * **注意点**: * Python/Selenium などの提案が来る場合があるが、PowerShell 単体での実現も可能。 * ログイン操作、MFA、更新操作（申請・承認など）は自動化範囲外とするか、別途対応を検討。認証情報はスクリプトに含めない。 ### 5. 実行・修正をループ * **進め方**: 1. PowerShell スクリプトを実行。 2. 結果を確認し、エラーや問題があればCopilotに状況を説明して修正。 3. 要件を満たすまで、追加・修正・テストを繰り返す。 * **ポイント**: 最初から完璧を目指さず、小さく始めて徐々に機能追加する。 ### 6. できあがったもの * 自動化された処理のGIF動画が示されており、実際の動作を確認できる。 * 使用コードは記事の最後に掲載。 ### 7. 応用編 (VBA との連携) * PowerShell スクリプトを Excel VBA と組み合わせることで、運用を容易にする。 * Excel 上でコード一覧を表示し、実行用バッチファイルを作成して実行する。 ### 8. M365 Copilot (Premium) を使える場合 * **機能強化**: * Excel Copilot のワークブック編集機能で、シート内容を見ながら修正指示が容易に。 * Work IQ による社内情報（マニュアル等）を参照した自動化方針の策定やコード生成。 * **Excel 完結構成のイメージ**: * PowerShell コードはシートに保持。 * 実行用バッチファイルもシートに保持。 * 結果シートでCSVの取り込みやログ確認。 * VBA ボタンでシート内容をファイルに書き出して実行。 * **利点**: デバッグや試行錯誤を Excel 内で完結させやすい。 ### 9. 最後に * 制約のある環境でも、AI 活用は十分に可能。 * フル自動化でなくとも、「ちょいRPA」で確認業務などを自動化し、小さく始めることを推奨。 * AI 活用による「自動化できている」というインパクトを出すことで、さらなる AI 導入への突破口となる。 * コードのレビューや、レビュー工数削減のための PowerShell レビューエージェントの検討も示唆。 ### 10. サンプルコード * PowerShell のみで動作する、UIオートメーションによるデモサイトのステータス確認スクリプトが提供されている。 * ブラウザ起動、ボタンクリック、テキスト抽出などの機能が含まれている。 ### 11. 参考リンク * Microsoft 365 Copilot および PowerShell に関する公式ドキュメントや関連Qiita記事へのリンクが掲載されている。
