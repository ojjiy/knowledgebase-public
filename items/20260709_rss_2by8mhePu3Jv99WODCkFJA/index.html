AWS Builder Center 新機能 - 無料サンドボックスで Claude Code ワークショップを試してみた

Date: 2026-07-09

Source: rss / DevelopersIO

Original URL

AWS Builder Center は、2026年7月8日より、一部のワークショップで無料サンドボックス環境の提供を開始しました。これにより、これまで必要だった自前のAWSアカウントが不要になり、Builder IDのみでハンズオン演習に参加できるようになりました。 この無料サンドボックス環境は、AWS Builder ID を持っていれば誰でも利用可能で、利用したAWSリソースに対する費用は一切かかりません。ただし、サンドボックス環境のリクエストは週に1回、利用時間は準備完了通知から8時間という制限があります。 2026年7月9日時点で、無料サンドボックス環境が利用可能なワークショップは7件でした。これらのワークショップの多くは、生成AIやAmazon Bedrockに関連するもので、今後対象ワークショップは拡大される予定です。 筆者が試した「Amazon Bedrock上のClaude Code」ワークショップでは、サンドボックス環境のリクエストから準備完了まで約22分かかりました（案内の約15分より若干長め）。環境が準備されると、Workshop Studio に遷移し、AWSマネジメントコンソールへのアクセスやAWS CLIの一時認証情報が提供されました。ワークショップの内容は日本語にも対応しており、チームでのClaude Codeのセットアップ、ワークフローの確立、高度なワークフローの実装、モニタリング・コスト管理、Amazon Bedrock AgentCoreでのClaude Agent SDKのデプロイまで、網羅的に学ぶことができます。 この無料サンドボックス環境は、AWSの最新技術を気軽に試したい開発者にとって非常に有用なサービスと言えます。
