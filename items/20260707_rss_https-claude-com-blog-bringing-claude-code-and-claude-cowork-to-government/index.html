Bringing Claude Code and Claude Cowork to government

Date: 2026-07-07

Source: rss / Claude

Original URL

Anthropicは、政府機関向けに「Claude Code」と「Claude Cowork」をパブリックベータ版として提供開始した。これらは、商用版と同じアプリケーションを基盤とし、FedRAMP High認証済みの環境で提供される。 「Claude Code」により、公共サービスを支えるソフトウェアシステムの構築や近代化が可能になる。「Claude Cowork」は、デスクトップ上のファイルと直接連携し、メモ作成、RFPレビュー、ケースワーク、プレゼンテーション資料作成などをClaudeに委任できる。 今回の拡張には、追加のガバナンス機能も含まれる。管理者は、設定のデフォルト値を設定したり、部署間で利用状況を配分・管理したりできる。セキュリティチームや承認担当者は、改ざん防止の監査ログや、機関のATO（Authority to Operate）プロセスをサポートするドキュメントを利用できる。 主な新機能と利点は以下の通り： * **迅速な機能提供:** 政府機関は、商用ユーザーと同じペースで新機能を利用できる。 * **ローカルでの履歴保存:** 会話履歴は、機関が管理するデバイス上にローカルに保存される。 * **FedRAMP High環境での推論:** 推論処理は、FedRAMP High認証済みの環境で実行される。 * **予算に合わせた請求:** プログラムオフィスは、AI利用費用を予算に紐づけることができ、固定座席数または独自の座席階層と利用制限を設定できる。利用状況は固定単位で購入され、上限額が設定されている。管理者は、管理者コンソールでユーザーごと、モデルごとの利用状況を追跡でき、残高が少なくなる前に自動アラートが通知される。 * **組織構造に合わせた管理:** 部門レベルの管理者は、下位機関に座席数と前払い利用枠を割り当て、各機関が独自のユーザーを管理できるようにできる。SCIMグループマッピングを使用して、特定の座席階層に対してレート制限、金額上限、許可モデルを設定できる。また、階層化された設定により、Claudeが接続できるもの、利用可能な機能、Claudeとユーザーとの対話方法をガイドする指示など、下位機関向けのデフォルト設定が可能となる。 * **設計による監視:** すべての管理アクションは、ハッシュチェーン化された監査ログに記録され、組織管理者は製品内で直接確認できる。Anthropic側の機密性の高い操作には、2人による承認が必要となる。利用状況のエクスポートは、メーターデータのみで行われるため、機密情報を移動させることなく、ATOやIG（Inspector General）からの要求に対応できる。 セキュリティチームは、FedRAMP Secure Configuration Guideを公開しており、Claude for Government製品を安全に設定するために利用できる。また、FedRAMP要件に基づき、変更通知とペネトレーションテストの概要が提供される。これらは、AnthropicのTrust Centerを通じてNDAの下で入手可能。アプリケーションは、標準的な機関のMDMプラットフォームを通じて展開できる。 Claude for Governmentは、本日よりベータ版として利用可能。Anthropicが契約と請求を継続するため、政府機関は別途クラウドプロバイダーとの関係を構築する必要はない。新規顧客は、claude.com/solutions/government からアクセスをリクエストできる。セキュリティチームは、指定されたリンクからペネトレーションテストの成果物をダウンロードできる。
