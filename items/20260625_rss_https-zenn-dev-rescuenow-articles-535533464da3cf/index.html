ローカルのGit差分をGitHub風UIでレビューできる「difit」を使ってみた

Date: 2026-06-25

Source: rss / Hatena Bookmark Programming

Original URL

「difit」は、ローカルのGit差分をGitHub風のUIでブラウザ表示・レビューできるCLIツールです。 **主な機能・特徴:** * **GitHub風の差分ビュー:** シンタックスハイライト付きで、行番号付きの差分を分かりやすく表示します。 * **コメント機能:** 差分の各行にレビューコメントを追加でき、コメントには「Copy Prompt」ボタンが付いています。 * **AIエージェント連携:** 「Copy Prompt」ボタンで、Claude CodeなどのAIコーディングエージェントに修正依頼を渡すためのプロンプトをコピーできます。 * **多彩な比較モード:** 単一コミット、ブランチ間、作業ディレクトリ、ステージング、GitHub PRなど、様々な比較対象を指定できます。 * **コメントの永続化:** コメントはブラウザのlocalStorageにコミット単位で保存されます。 * **自動折りたたみ:** 削除ファイルや自動生成ファイルは折りたたんで表示され、見やすさを向上させます。 * **Skills連携:** AIコーディングエージェントからdifitを呼び出し、差分レビューを依頼する連携も可能です。 **利用シーン:** * PRを作成するほどではないが、差分をきちんと見返したい場合。 * AIに差分レビューを手伝ってほしい場合。 * コミット前のセルフレビュー。 **利用上の留意点:** * ブラウザを開くことが前提のため、SSH先サーバーなどではオプション指定が必要です。 * コメントはローカルのlocalStorageに保存されるため、チーム間での共有には向きません。 **インストール・起動:** * `npx difit` でインストール不要で実行できます。 * `npm install -g difit` でグローバルインストールも可能です。 * リポジトリのディレクトリで `npx difit` を実行すると、最新コミットの差分がブラウザで開きます。 **比較モードの例:** * `difit HEAD`: 最新コミットの差分 * `difit .`: 未コミットの変更をすべて表示 * `difit staged`: ステージング済みの変更を表示 * `difit --pr <PR_URL>`: GitHubのPRをレビュー 「difit」は、ローカルのGit差分確認作業を効率化し、AIとの連携も容易にする便利なツールです。
