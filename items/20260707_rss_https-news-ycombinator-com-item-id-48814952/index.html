Show HN: Fast, native Mac file manager (filters, fuzzy find, 9 MB, no Electron)

Date: 2026-07-07

Source: rss / Hacker News Front Page

Original URL

「WhimFiles」は、macOS向けの高速でネイティブなファイルマネージャーです。ファイルの種類、日付、サイズで絞り込み、フォルダを一度に狭めることができます。あいまい検索によるフォルダジャンプ、ホバープレビュー、2つのフォルダを並べて表示する機能などを備えています。Electronを使用せず、アプリサイズは約9MBと軽量です。 主な機能は以下の通りです。 * **強力なフィルター:** ファイルの種類、日付、サイズでリアルタイムに絞り込み可能です。画像、PDF、動画、ドキュメントなどのプリセットカテゴリや、カスタム拡張子によるフィルターも定義できます。日付範囲、サイズ制限、検索語を同時に設定でき、入力と同時に結果が更新されます。 * **デュアルペイン:** 2つのフォルダを並べて表示し、直接作業できます。右クリックまたはコマンドパレットでファイルをコピーまたは移動できます。各ペインは独立したフォルダ、フィルター、検索、履歴を持ちます。 * **高速な移動:** 「あいまいフォルダ移動」（⌘G）機能により、パスの数文字を入力するだけで目的のフォルダにジャンプできます。ファイルへのジャンプ（⌘P）も可能です。サイドバーにはブックマークとマウントされたボリュームが表示され、パンくずリストで親フォルダに簡単に移動できます。 * **プレビュー機能:** 画像やPDFファイルにホバーすると、クリックせずに大きなプレビューが表示されます。ファイルアイコンをクリックするとプレビューが表示され、アイコンプレビューモードを切り替えることでサムネイルを直接ファイルリストで確認できます。スペースキーでネイティブなクイックルックが起動し、矢印キーでリスト内のファイル間を移動できます。 * **タブ機能:** 必要なだけタブを開き、それぞれがフォルダ、フィルター、デュアルペインレイアウトを記憶します。タブ間を素早く移動したり、再起動後にタブの状態を復元したりできます。 * **ファイル操作:** 一括リネーム（検索・置換、連番、接頭辞/接尾辞、大文字/小文字変更）、HEIC、WebP、AVIF画像をJPGまたはPNGに変換、ZIP圧縮、移動やゴミ箱操作の取り消し、Finderスタイルの競合処理によるカット、コピー、ペーストが可能です。 * **ターミナル連携:** サイドバーにはブックマークフォルダやマウントされたボリュームが表示されます。コマンドパレット（⇧⌘A）は、すべての一般的なアクションにキーボードからアクセスできます。カスタムファイルタイプフィルター、フォルダサイズの再帰的計算、隠しファイルの表示、フォルダを優先表示する設定があります。また、コマンドラインからWhimFilesを起動するための簡単なシェル関数を追加できます。 * **対象ユーザー:** ファイル管理、コード管理、または単にファイルを見つけにくいと感じるユーザーに適しています。特に、多数のファイルがあるダウンロードフォルダの整理、画像やPSDなどのアセット管理、コード編集時のプロジェクトディレクトリ間の移動、写真コレクションのブラウジング、メディアフォルダの整理などに役立ちます。 WhimFilesは、30日間の無料トライアルが提供されており、買い切り型のライセンスで購入できます。macOS 12以降およびApple Silicon（M1以降）に対応しています。
