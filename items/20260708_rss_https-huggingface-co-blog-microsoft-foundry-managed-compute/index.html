Hugging Face Models on Foundry Managed Compute

Date: 2026-07-08

Source: rss / Hugging Face Blog

Original URL

Microsoft Azure Foundry Managed Compute 上の Hugging Face モデルは、Hugging Face エコシステムから厳選されたオープンウェイトモデルを、週次で更新されるカタログで提供し、ワンクリックで Foundry Managed Compute にデプロイできるようにするサービスです。これらのモデルは Azure 上に事前にステージングされ、Microsoft によって実行環境が構築・スキャンされており、Foundry 上の他のモデルと同様のエンタープライズレベルのセキュリティ、ガバナンス、オブザーバビリティ、請求機能が提供されます。 Foundry は、エージェント型 AI アプリケーションの構築および運用プラットフォームであり、Microsoft、OpenAI、Anthropic、Meta、Mistral、DeepSeek、Hugging Face など、幅広いモデルを単一のエンドポイントと SDK で利用できます。Agent Service は、マルチエージェントのオーケストレーション、メモリ、Foundry IQ による知識グラウンディング、ツール連携機能を提供し、エンドツーエンドのトレーシング、リアルタイム監視、継続的な評価、プロンプト最適化などのオブザーバビリティ機能も備えています。 Foundry Managed Compute は、Foundry 上でのモデルデプロイオプションの 3 つ目として、オープンソースおよびカスタムモデル向けのマネージド GPU プラットフォームサービスです。モデルのパラメータ数、コンテキスト長、レイテンシーまたはスループットの最適化などの要件に基づいてデプロイインスタンスを設定でき、Foundry が GPU トポロジーを管理します。vLLM、SGLang、TensorRT-LLM、NIM、TEI、llama.cpp などのサポートされる実行環境では、コンテナの更新、ランタイムのアップグレード、セキュリティパッチが自動的に適用されます。 Hugging Face は、1,500 万人の開発者、40 万組織、300 万以上のオープンモデルが集まるプラットフォームであり、最新の AI 技術が毎週追加されています。オープンモデルは、ベンチマークでプロプライエタリモデルに匹敵する性能を示し、ファインチューニング、蒸留、量子化、LoRA によるカスタマイズが可能です。モデルの重みはユーザーのテナント内で実行され、インフラストラクチャを制御できます。 Hugging Face Models on Foundry は、Hugging Face のオープンモデルをエンタープライズグレードのサービスとして提供するための運用レイヤーを提供します。Foundry モデルカタログには、最新のトレンドモデルが毎週追加され、テキスト、ビジョン、オーディオ、マルチモーダルなど、あらゆるモダリティをサポートします。モデルは Safetensors 形式のみで、セキュリティスキャン済みであり、信頼できないコードの実行は排除されます。モデルには、vLLM、SGLang、TensorRT-LLM、NIM、TEI、llama.cpp、hf-serve などの適切な実行環境が自動的に選択され、デプロイされます。 モデルのデプロイは、カタログからモデルを選択し、デプロイテンプレート（レイテンシーまたはスループット最適化、アクセラレータファミリー、コンテキスト長などを指定）を選択し、インスタンス数を設定してデプロイする 5 ステップで行われます。デプロイされたモデルは、統一された Foundry エンドポイントを通じて、SDK（OpenAI SDK など）でスコアリングできます。Foundry Agents では、Collection からのチャットコンプリーションモデルは、管理者接続モデルとして統合され、同様の SDK で呼び出すことができます。 現在プレビュー版では、Foundry モデルカタログに Hugging Face Collection が提供されており、多様なモダリティの数千のモデルが利用可能です。NVIDIA A100、H100、AMD MI300X などのアクセラレータにデプロイでき、グローバルおよびデータゾーンスコープで利用できます。今後は、Hugging Face エコシステムのより広範なカバー、追加のアクセラレータファミリー、および Bring Your Own Weights のサポートがロードマップにあります。Microsoft Foundry は、オープンモデルをエンタープライズで運用するためのプラットフォームとして、キュレーションされ、ライセンス・セキュリティスキャン済みのモデル、CVE スキャン済みの実行環境、およびエンタープライズグレードの管理機能を提供します。
