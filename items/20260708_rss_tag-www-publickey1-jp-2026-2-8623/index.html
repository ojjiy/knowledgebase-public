Visual Studio Code、従量課金制になったGitHub Copilotの使用料を表示可能に（訂正あり）

Date: 2026-07-08

Source: rss / Publickey

Original URL

Visual Studio Codeに、従量課金制となったGitHub Copilotの利用状況を表示する新機能が追加されました。 GitHub Copilotは2026年6月より、チャットやAIエージェント利用時に消費したトークン数に応じて課金される従量課金制に移行しました。そのため、ユーザーは利用料金が想定を超えないよう注意が必要となります。 今回追加されたVisual Studio Codeの新機能では、以下の情報がリアルタイムで表示されます。 * **トークン消費量**: 現在の利用状況に対応する利用料金がリアルタイムで表示されます。 * **アラート通知**: 事前に設定した上限金額に近づいた場合や、上限に達した場合にアラートが表示されます。 この機能は、2026年6月17日にリリースされたVisual Studio Code 1.125で提供開始されました。
